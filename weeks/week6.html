End of week reflection: This week I've learned how to use Express.js to build web applications with Node.js. The framework makes it much easier to create routes, handle requests, and render views compared to using the built-in HTTP module. I particularly enjoyed working with middleware and learning about the MVC pattern for organizing code. I now have a good understanding of how to build simple web applications with Node.js and Express.
