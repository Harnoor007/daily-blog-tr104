Started the week learning about databases and how to use them with Node.js applications.

Topics covered:

  • Introduction to Mongoose
  • Creating schemas and models
  • Validating data with Mongoose
  • CRUD operations with Mongoose
  • Mongoose middleware

Practical exercises:

  • Set up Mongoose in an Express application
  • Created schemas for different data types
  • Implemented data validation
  • Built a simple API with Mongoose models
