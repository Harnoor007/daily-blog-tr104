Day 1 - Monday

This week we're continuing with more advanced JavaScript topics. Today's focus was on arrays and their manipulation methods.

Topics covered:

  • Array creation and initialization
  • Accessing and modifying array elements
  • Array methods: push(), pop(), shift(), unshift()
  • Array methods: splice(), slice()
  • Iterating through arrays with loops

Practical exercises:

  • Created and manipulated arrays of different data types
  • Implemented common array operations (adding, removing, modifying elements)
  • Built a simple to-do list application using arrays

Day 2 - Tuesday

Today we explored higher-order array methods which are crucial for modern JavaScript development.

Topics covered:

  • forEach() method for iteration
  • map() for transforming arrays
  • filter() for selecting array elements
  • reduce() for cumulative operations
  • find() and findIndex() for searching
  • some() and every() for testing conditions

Practical exercises:

  • Refactored previous array exercises using higher-order methods
  • Built a data processing pipeline using chained array methods
  • Created a filtering system for a mock e-commerce product list
